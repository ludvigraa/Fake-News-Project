# Fake News Detection — GDS Final Assignment

This project builds and evaluates fake news detection models using the FakeNewsCorpus (995,000 articles) and the LIAR dataset. It covers preprocessing, a logistic regression model, an advanced SVM model, and cross-dataset evaluation.

## Large Files

The following files are not included due to size:
- `995,000_rows.csv` — the raw FakeNewsCorpus subset. Download from [FakeNewsCorpus](https://github.com/several27/FakeNewsCorpus) and place in the root directory.
- `processed_data/fakenews_995k_processed.csv` and the train/val/test splits — generated by running Part 1.

The trained model `.pkl` files in `models/` are included and can be used to reproduce Part 4 results without retraining.

## How to Reproduce Results

Install dependencies:
```
pip install pandas numpy scikit-learn matplotlib joblib nltk
```

### Part 1 — Preprocessing
Run all cells in `FakeNews_Part1_2.ipynb` up to and including the Task 4 split section. This requires `995,000_rows.csv` in the root directory.

### Part 2 — Logistic Regression
Continue in `FakeNews_Part1_2.ipynb` from the Part 2 section (cells 46 onwards). Skip the GridSearchCV cell — best parameters are already hardcoded in the fit and save cell.

### Part 3 — Linear SVM
Run all cells in `FakeNews_Part3.ipynb` in order. Hyperparameter tuning (Section 3) tests 16 combinations and may take 20–40 minutes.

### Part 4 — Evaluation
Run all cells in `FakeNews_Part4.ipynb` in order. The saved `.pkl` files in `models/` and the processed data splits in `processed_data/` are required.

## Label Scheme
- 0 = fake: `fake`, `conspiracy`, `junksci`, `rumor`, `clickbait`, `unreliable`
- 1 = reliable: `reliable`
